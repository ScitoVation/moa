List of classification parameters provided in the toolbox

Several classification parameters are provided in the toolbox in order to evaluate the classification performances both in fitting and in cross-validation. The classification parameters are provided as MATLAB structure (class_param) by the routines model_cpann, model_skn, model_xyf, and cv_cpann, cv_skn, cv_xyf. In the first case, the results refer to the classification estimation in fitting, in the second case in cross-validation. The classification parameters can be analysed by typing, in the MATLAB command window:

model.res.class_param

when fitting the model, i.e. with the routine model_cpann, or

cv.class_param

when cross-validating the model, i.e. with the routines dacv, plsdacv, knncv and cartcv.

ROC curves

ROC (Receiver Operating Characteristics) curves are graphical tools for the analysis of classification results. ROC curves can be calculated only in the graphical interface of the Kohonen and CP-ANN toolbox. A ROC curve is a graphical plot of sensitivity and 1-specificity, for a binary classification system as its discrimination threshold is changed.
A single value of sensitivity and 1-specificity can be calculated from a contingency table and consequently each contingency table represents a single point in the ROC space. For each threshold value, a classification rule is calculated and the respective contingency table is obtained. The best possible classification method would yield a point in the upper left corner of the ROC space, representing maximum sensitivity and specificity, while a random classification give points along the diagonal line from the left bottom to the top right corners. Summarsiing, ROC curves are calculated for each class, separately, by changing the threshold of assignations of output neurons to the considered class from 0 to 1.
