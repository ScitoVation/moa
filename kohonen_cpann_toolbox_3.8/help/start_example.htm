Note that the first method always assigns neurons to a class, while the other three methods can give not assigned neurons; in these cases, the class of samples placed in not-assigned neurons won't be recognized.

The proposed thresholds can be changed b y the user in the model_cpann routine. The default values are:
thr_assign(1) = NaN;
thr_assign(2) = 0.3;
thr_assign(3) = 0.5;
thr_assign(4) = 0.5;
