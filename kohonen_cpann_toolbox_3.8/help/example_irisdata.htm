The IRIS dataset

IRIS dataset is a classical data set used by statisticians and chemometricians to check classification methods. It is composed by 150 samples of flowers divided into 3 classes (50 setosa, 50 versicolor, 50 virginica) and described by 4 variables (sepal length, sepal width, petal length, petal width). It has been published by Fisher in the following paper:

Fisher RA. (1936) The use of multiple measurements in taxonomic problems. Annals of Eugenics 7 179-188.
