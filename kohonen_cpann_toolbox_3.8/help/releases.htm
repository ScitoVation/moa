What's new in version 3

In the toolbox version 3, two new supervised methods based on Kohonen maps have been added: Supervised Kohonen networks (SKN) and XY-fused Networks (XY-F). Have also a look at the theory section for info on these methods.

Beside the sequential training, the algorithm for the batch training of the weights has been introduced. Details are given here.

The hexagonal map topology has been added. It can be defined in the setting structure or directly in the GUI interface. The Kohonen weights can be initialised both randomly or on the base of the eigenvectors corresponding to the two largest principal components of input data. Details are given here.

A new strategy for the selection of the optimal number of epochs and neurons of supervised models is introduced in the toolbox version 3. This strategy exploits the ability of Genetic Algorithms to optimize network parameters. Have a look here for further info.

External samples (not used to build a model) can now be seen in the top map of the GUI interface. If a class vector is loaded for external samples, the corresponding classification parameter are calculated. ROC curves are calculated on each class by changing the threshold over the output weights. In the PCA analysis on the Kohonen weights, it is possible to draw Voronoi regions on the PCA score space. Prediction of external samples can be saved to the matlab workspace as matlab structures (see here for details).

The calculation of the error rate and non-error rate of the model was changed. The time consuption for the calculation of models has been 50% reduced.

A form for plotting boxplots, histograms and biplots of variables has been added in the "view" menu of the GUI.


What's new in version 2

In the toolbox version 2, a GUI graphical interface has been added. The graphical interface enables you to do all the steps of the analysis (data loading, seting preparation, model calculation, sample prediction, cross-validation). If you wish to use the graphical interface, read the GUI help section.

Data can now be scaled. However, the scaled data are then always range scaled inbetween 0 and 1, in order to make them comparable with the net weights. Look the setting section for further information.

In the setting structure, the field "a_chg" has been removed. It was the settings for deciding if the learning rate was proportional to epochs ('to_epo') or epochs*samples ('to_epoobj'); the default value now is always set to 'to_epo'.

Two new fields have been added in the setting structure: "show_bar" enables the user to see (not see) a waiting bar during the calculation of the model; "absolute_range" enables the range scaling to be classical (separatly on each column) or absolute (with the absolute maximum and minimum values of the dataset): read the how to start section, where a graphical example on the absolute range scaling is shown.

When calculating (and cross validating) Counterpropagation ANNs, the class weights are now saved. Look the CP-ANNs section for further details.

In the GUI for visualizing the results, now it's possibile to calculate Principal Component Analysis on the Kohonen weights. Look the corresponding help section for further info.

Now cross-validation (only for CP-ANNs) can be performed both with venetian blinds and contiguous blocks.

An extended explanation of the classification parameters (such as error rate, non-error rate, specificity, sensitivity, etc...) has been added in the help section, here.
